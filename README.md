Mi CV Web - Portfolio Personal

Este proyecto consiste en el desarrollo de mi currículum profesional en formato de página web. El objetivo es presentar mi perfil como diseñadora de una manera interactiva, clara y moderna. El proyecto está realizado en Visual Code, con una estructura de HTML5 y CSS. Este mismo presenta con un Head, Body y Footer con sus respectivas etiquetas dentro para que el armado de la página sea semánticamente correcta y en CSS, se utilizó la herramienta Flexbox.
El diseño de la página web está basado en mi marca personal "AGUSPINTHA" y su respectiva identidad visual. La tipografía utilizada es EncodeSans donde se utilizaron las variables de la misma.
Este proyecto se encuentra en estapa de desarrollo (preentrega)
